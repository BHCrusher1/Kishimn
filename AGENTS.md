# AGENTS.md

## プロジェクト概要

- FFmpeg フロントエンドツール "Kishimn"
- FFmpegをコマンドラインではなくGUIで簡単に扱えるようにするためのフロントのみのアプリ
- FFmpegは本アプリに同梱しない

## 使用技術

- .NET 10
- WinUI 3
- FFmpeg 8.1

## 動作環境

- Windows 11 x64
- 依存関係は使用するユーザーPCにインストールされている前提とする

## 必須要件(Must)

- C# XML ドキュメントコメントに従い日本語でコメントを追加する
- 画面項目の選択肢内容はUI(xaml)側ではなく全てコード(cs)側でまとめる
- コードの規約・フォーマットは .editorconfig に従う

## 推奨要件(Should)

- ユーザーに配布するファイル一式のサイズを可能な限り小さくする
- コードを変更したらビルドを行い、ビルドエラーが起こらないこと、アプリケーションがエラー無く起動することを確認する

## ビルドコマンド

`dotnet publish -c Release -p:Platform=x64 --self-contained false`
